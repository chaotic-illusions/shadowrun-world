>> LOADOUT · HOT ZONE + MEMORY BARS

Throwaway prototype · v11 memory bars + nested hot zone + fixed-size bordered cards. Not wired to the live builder.
The two bars are the v11 memory map: every program is a colored segment sized by its Mp, so you read fit at a glance. Below, the glowing violet frame is Active, drawn inside the Storage board — an active program also occupies storage memory, so it counts against both meters and can never live in active alone. It appears once, in the Active frame; it is not repeated as a card in the reserve pool below, but the storage meter and bar still include its size. Drag a card between Library, the reserve pool and the Active frame to move it; hover a card to light its segment(s) in both bars. A program rated above the deck's MPCP can't be loaded onto the deck at all — not active, not storage (your iron can't run it, and you can't swap it in later) — so dropping it anywhere on the deck is rejected.
